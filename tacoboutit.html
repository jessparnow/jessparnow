Taco 'Bout It

Description

Users can create an account and add reviews. Styling done by Mariam. Backend by both Mariam and Jess. App is currently under construction.

Technologies Used:

  • MYSQL as a database
  • Sequelize ORM
  • Passport for authentication
  • ESLint
  • Travis CLI

Features to be added:
